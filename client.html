Moonlight Web Proxy
A WebSocket bridge between this offline page and your moonlight-web-stream server.
1
Start proxy-server.js on your local machine
node proxy-server.js
2
Enter the proxy WS address and your moonlight-web-stream server URL (the Rust server running on port 8080)
3
Click Connect. The moonlight UI will load inside this page and all traffic tunnels through the proxy.
